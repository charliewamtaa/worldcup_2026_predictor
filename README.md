So around 2002, I came across an Excel sheet floating around that was laid out like the World Cup schedule. The only interactive part of the spreadsheet was adding the scores — everything else updated itself. The group stages, round of 16, all the way to the final. That experience was genius to me. I'd play around with different score scenarios, seeing which team would play which in the knockouts, trying to see if my team would reach the final.

This inspired me to use Claude to develop the same Excel sheet. Later, I thought — why not just have Claude do an HTML page of the same thing? And here is the result. I've made some edits along the way. Didn't want to overcomplicate it, so I kept it as a static HTML page with no backend. Who knows, it might inspire other ideas further down the line for different tournaments.

Please feel free to suggest any other feature I can add to this.
